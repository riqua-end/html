overflow 프로퍼티

overflow에 hidden 값을 적용하면 박스를 넘어가는 내용이 잘려 보이지 않습니다. 잘린 내용이 있다는 표시는 text-overflow: ellipsis속성을 사용하면 ...으로 표시됩니다.


overflow에 visible 값을 적용하면 콘텐트가 박스를 넘어가서도 출력됩니다(overflow의 디폴트)


overflow에 scroll 값을 적용하면 박스에 스크롤바를 붙여 출력합니다.(넘치지 않아도 스크롤을 만들어 줍니다)


overflow에 auto값을 적용하면 박스에 스크롤바가 필요시에 스크롤바를 붙여 출력합니다.

박스의 height없을시 오버플로우가 일어나면 어떻게 될까 살펴 봅시다. 박스의 높이가 지정 안되면 자동으로 높이가 조절 되어 넘치지 않습니다.
